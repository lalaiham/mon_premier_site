Le web dynamique

Il existe 2 types le sites différents. Le web statique et le web dynamique.

Sur ce site , on va vous parler plus précisément du web dynamique.

Un site dynamique suit un objectif concret : celui d'adapter les informations qui sont affichées en fonction de l'utilisateur qui demande à les voir. Son affichage et les informations partagées dépendront ainsi d'un éventuel formulaire rempli par l'internaute, de ses préférences et publicités ciblées… Cela signifie que les utilisateurs n'auront pas tous le même aperçu de la page web dynamique. Par exemple, Facebook est un site internet dynamique, qui change et s'actualise en fonction des dernières publications et des préférences de l'utilisateur.

- Langagues qui permettent de rendre un site dynamique

Quel que soit le type de site web que vous souhaitez créer, HTML et CSS sont donc indispensables puisque ces langages sont la base de tous sites web. Cependant, ils ne suffisent pas pour la création de sites dynamiques. Il faut les compléter avec d’autres langages. 
  • PHP : Langage que seuls les serveurs comprennent et qui permet de rendre votre site dynamique. C’est lui qui « génère » la page web. Il peut fonctionner seul mais il est plus efficace quand il est combiné à MySQL
  • MySQL :Système de gestion de bases de données. Il enregistre et organise les données (liste des membres, messages sur le blog…) pour vous aider à les retrouver plus facilement.

  • Catalogue en ligne :le client sélectionne ses paramètres  (taille, couleur, prix…). Selon ses paramètres, une page « sur mesure » est créée. 
  • Les jeux en ligne
  • Les moteurs de recherche les pages de résultats qu’ils proposent sont adaptées aux recherches des internautes.

    • - Lien entre le client et le serveur

    Les sites Web dynamiques utilisent les scripts côté client pour préparer une conception dynamique et du code côté serveur pour gérer les événements, gérer les sessions et les cookies, et stocker et récupérer les données de la base de données.
